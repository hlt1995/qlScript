# 青龙教程

- `caiyun-1.0.0-alpha.7.mjs`：和彩云脚本
- `caiyun.ql.mjs`：青龙启动脚本
- `asign.json`：配置文件

## 青龙脚本配置

```
node caiyun.ql.mjs
```
> 青龙任务命令


```
30 9,16,23 * * *
```
> 青龙任务定时，每天9:30,16:30,23:30运行一次脚本
